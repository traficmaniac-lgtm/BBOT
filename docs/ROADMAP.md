# Roadmap

## Milestone 0.1 — Foundation (текущая)
- [x] GUI-скелет с вкладками Dashboard, Pairs, AI, Trading, Risk, Logs, Settings.
- [x] Конфиги: загрузка/сохранение YAML, активный конфиг в UI.
- [x] Мок-обновление списка fee-free пар и выбор активной пары.
- [x] AI-клиент (OpenAI или мок) с валидацией JSON.
- [x] State machine и статусы.
- [ ] Реальный торговый движок (зарезервировано).

## Milestone 0.2 — Trading core
- Реализация обменного клиента (Binance/OKX) с реальным запросом пар.
- Оркестратор стратегий, тестовый симулятор сделок.
- Настройки рисков и ордер-менеджмент.
- Базовый репортинг и экспорт результатов.

## Milestone 0.3 — Strategy packs
- Поддержка нескольких стратегий (trend-following, grid, DCA).
- Бэктестинг и валидация на исторических данных.
- Подсказки AI для выбора стратегии.

## Milestone 0.4 — Production hardening
- Расширенный логирование + алерты.
- Шифрование секретов в конфиге, secure storage.
- Автообновление приложения, проверка версии.

## Milestone 0.5 — UX polish
- Темизация, hotkeys, настройки локализации.
- Улучшенный мониторинг позиций и PnL.
- User onboarding и туры.
